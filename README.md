# CVSS 3.1 Calculator

Implements the CVSS 3.1 calculation as specified by FIRST [1].

1. Define the input values in `score_entry.py`
2. Run the calculation:

~~~sh
$ python3 cvss_calculator.py
Base Score: 7.1
Temporal Score: 7.1
Environmental Score: 7.2
~~~

[1] https://www.first.org/cvss/v3.1/specification-document
